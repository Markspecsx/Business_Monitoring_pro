# Business Suite MVP

A complete, offline-ready business financial management web app that tracks projects, allocates profits, monitors daily expenses, and visualizes exactly where every dollar goes.

## Features

- **Monthly Monitoring** – Track client projects, revenue, and costs with auto-calculated net profit
- **Profit Allocation** – Split net profit into 8 budget categories (40% Necessities, 10% Savings, 10% Debt, etc.)
- **Daily Expenses** – Log spending with real-time budget progress bars and category tracking
- **Money Flow Visualizer** – See exactly where every dollar is allocated, spent, and remaining
- **Supplier Payments** – Track vendor invoices, payments, and outstanding balances
- **Debt Repayment** – Manage loans, credit cards, and track payoff progress
- **Client Management** – Store client info, payment methods, and monthly sales history

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- LocalStorage for data persistence (no backend required)
- Fully responsive (mobile + desktop)
- CSV export for all tables

## Quick Start

1. Download `index.html`
2. Open in any modern browser
3. Start tracking your finances immediately

No installation, no server, no internet required after download.
